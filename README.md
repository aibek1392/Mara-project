# Door 2 Door — интернет-маркетплейс

Маркетплейс с поддержкой **3 языков**: Русский (RU), English (EN), Кыргызча (KY).

Переключатель языка — в шапке сайта (кнопки RU / EN / KY). Выбор сохраняется в браузере.

## Запуск

```bash
cd web
npm install
npm run dev
```

Откройте **http://localhost:5173**

## Демо-аккаунты (пароль: `123456`)

| Роль              | Email                    | Доступ                         |
|-------------------|--------------------------|--------------------------------|
| Владелец          | owner@door2door.com      | Панель владельца               |
| Покупатель        | user@door2door.com       | Покупки, корзина, аккаунт     |
| Продавец          | seller@door2door.com     | Кабинет продавца               |
| Менеджер доставки | shipping@door2door.com   | Панель владельца → Shipping Hub |
| Водитель          | driver@door2door.com     | Маршрут водителя `/driver`     |

## Связанные платформы (buyer → seller → shipping → driver)

1. **Покупатель** оформляет заказ на `/checkout` (имя, адрес, телефон).
2. **Продавец** в `/seller` отмечает сдачу товара на почтовый пункт → статус «Готов к забору».
3. **Менеджер доставки** в `/shipping` видит все заказы, сгруппированные по городу и адресу, принимает на склад и назначает водителя.
4. **Водитель** в `/driver` получает отсортированный маршрут и отмечает «В доставке» → «Доставлен».

## Страницы

- `/` — Главная
- `/shop`, `/shop/:category` — Каталог
- `/product/:id` — Карточка товара
- `/cart`, `/checkout` — Корзина и оформление
- `/search?q=...` — Поиск
- `/login` — Вход
- `/account` — Аккаунт покупателя
- `/owner` — Панель владельца
- `/seller` — Кабинет продавца
- `/owner` — Панель владельца (вкладка Shipping Hub для доставки)
- `/driver` — Маршрут водителя
